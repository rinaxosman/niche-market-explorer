# Niche Market Explorer

Niche Market Explorer is a simple web app that helps uncover and evaluate overlooked niche business ideas.

The app takes messy real-world input like Reddit posts, comments, or article snippets and uses AI to turn it into clean, structured market research.

## What it does

- Paste unstructured text (forums, notes, links)
- AI extracts niche business ideas
- Each idea includes startup cost estimates, demand signals, risks, and a simple score
- Ideas are saved and browsable in a clean dashboard

## Why it matters

Many good business ideas are scattered across the internet with no easy way to organize or validate them.  
This app turns inspiration into something actionable.

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma + SQLite
- OpenAI API

## Run locally

```bash
npm install
npm run dev
```
